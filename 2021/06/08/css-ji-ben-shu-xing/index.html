CSS基本属性

前端 CSS
css
发布日期:   2021-06-08
更新日期:   2021-06-15
文章字数:   1.4k

基本模型

emmet语法

1.写上标签按tap或者回车直接生成结构

2.快速生成标签 div*个数 例如 div*10按下回车或Tap生成10个div标签

3.父子标签
父>子*个数(个数可写可不写)
例如
table>tr*2>td*10
生成一个两行每行十格的单元格

4.兄弟标签
兄+弟

5.类名或id
标签.类名     p.qq--->  <p class="qq"></p>
标签#id       div#qq--> <div id="qq"></div>
不写标签标签默认为div

6.自增
标签加类名但是加一个$*
p.qq$*2

    <p class="qq1"></p>
    <p class="qq2"></p>

7.自带文字
q {干饭!} --->   <p>干饭!</p>

背景

1.背景颜色
background-color

2.背景图片
background-image url();
背景图片的平铺
background-repeat
{
    repeat
    norepeat 不平铺
    repeat-x 沿x轴平铺
    repeat-y 沿y轴平铺
}

3.背景图片位置
background-position: x y;
后面跟俩方位名词时不限顺序
background-position: center right;
在盒子的中右侧
名词有
left center right
top bottom
也可跟具体数字或者混搭
但一定注意顺序先x后y

4.背景固定
background-attachment: ;
scroll背景滚动
fixed背景固定

5.背景复合写法
background: 背景颜色 背景地址图片 背景平铺 背景图像滚动 背景图片的位置;

6.背景透明度
background rgba(0,0,0,a);
a取值0~1
0完全透明
1完全不透明

文本属性

1.text-align
水平对齐方式
left center right
垂直居中让行高
line-height等于width即可

2.text-decoration 
underline--下划线
line-through--删除线
overline--上划线


3.text-indent
首行缩进单位em/px
1em为一个字
2em首行缩进两个字符

4.line-height
行间距
行间距==字体大小+上下两间距
改行间距==改上下两间距

5.list-style
去点项目符号前的小圆点
属性为none

引入方式

内部标签
在head里写一个
<style>
选择器 {
    属性: 属性值;
}
</style>

行内样式
<h1 style="属性: 属性值;">
</h1>

外部样式
重新创建一个后缀为.css的文件
在.html的<head></head>中写入
<link rel="stylesheet" href="路径">
然后.css文件的书写方式为(直接这样写就行)
选择器 {
    属性: 属性值;
}

元素模式

1.块元素
<div></div>
<ol></ol>
<h1></h1>
<p></p>
<ul></ul>
<li></li>等
文字的内容不能加块元素
<p>
<div></div>
</p>
错误
p里面只能套行内元素或行内块不能套块
比如p里面套p


2.行内元素
相邻元素在一行上
不能直接设置宽和高
行内元素只能容纳行内元素或者文本


3.a中还能放块元素
<a>
<div></div>
</a>

4.行内块元素
具备块和行的特性
修改宽和高
一行多个
相邻有空隙

5.display: block;
属性加上后行元素边块元素
a {
    display: block;
}
display: inline
块元素转行元素
display: inline-block;
转化为行内块

字体属性

1.font-family
设置字体类型 宋体，楷体
可写多个如果前面的浏览器无法显示用后面的
每个字体用空格隔开


2.font-size
默认为16px 写单位px

3.font-weight
字体粗细
无单位
bold--粗--700
bolder--极粗
lighter--细
normal--400


4.font-style
normal--正常
italic--斜体

5.字体属性的复合
按照格式
font: font-style font-weight font-size/font-height font-family;
例子
font: normal 1000 16px/15px 宋体;
属性顺序固定用空格隔开
除了font-size 和 font-family必须写
其他可写可不写但是一定注意顺序

6.字体间距
letter-spacing

三种特性

1.层叠性
前提条件选择器相同
都是类选择或者标签选择
重复的样式设定遵循就近原则
离标签越接近(物理意义上)

2.继承性
子标签会继承父标签的一些性质
text
font
line
color
等文本样式
这样就可以处理一个大标签来处理相同的属性
例如给文本都设置为14px
可以让body属性标签改
body {
    font-size: 14px;
}


行高的继承:行高中若不写多少px写数字
则表示儿子大小的多少倍
如body中写行高为1.5
字大小为10px则继承行高为1.5*10

3.优先性
选择器不同看选择器权重   由小到大
*(通配符选择器)             0 0 0 0

元素选择器                  0 0 0 1

类选择器 伪类选择器          0 0 1 0

id选择器                    0 1 0 0

行内样式                    1 0 0 0 

!important                  正无穷
最后一个的使用方法是在属性中加入这句话
优先级直接起飞
p {
    color: red!important;
}
则优先级直接顶到最大

继承的权重最低
则解决了a链接不能通过改父辈元素性质而改变的问题
因为浏览器指定了a标签的属性为蓝色下划线
然后继承的权重最低

复合选择器会有权重叠加的效益
    <ol>
        <li></li>
        <li></li>
        <li></li>
    </ol>

ol li {

}

li {

}
则前面选择器权重大为0 0 0 1 + 0 0 0 1 == 0 0 0 2
但是注意不能进位
(0 0 0 1)*10=0 0 0 10 没有进位

    <ol class="qq">
        <li class="123"></li>
        <li></li>
        <li></li>
    </ol>

.qq li {}

.123 {}

.qq .123 {}
第一个选择器权重为0 0 1 1
第二个 0 0 1 0
第三个位 0 0 2 0
